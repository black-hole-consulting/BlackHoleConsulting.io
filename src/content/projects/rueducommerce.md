## Contexte

RueDuCommerce, l'un des pionniers du e-commerce francais, necessitait une evolution technique majeure pour accompagner sa croissance. Une mission de 3 ans pour transformer progressivement la plateforme tout en maintenant la continuite de service.

## Mon role

**PHP/MySQL Developer** puis **Lead Developer**, j'ai progresse au sein de l'equipe :

- Developpement de nouvelles fonctionnalites e-commerce
- Optimisation des performances du site
- Encadrement technique de l'equipe
- Migration vers des architectures plus modernes

## Realisations

### Phase 1 : Developpement (2008-2009)

- Implementation de fonctionnalites catalogue
- Developpement du tunnel d'achat
- Integration des moyens de paiement
- Corrections et ameliorations continues

### Phase 2 : Optimisation (2009-2010)

- Audit et optimisation des requetes SQL
- Mise en cache strategique
- Amelioration des temps de chargement
- Refactoring du code legacy

### Phase 3 : Leadership (2010-2011)

- Encadrement d'une equipe de developpeurs
- Definition des standards de code
- Revues de code et mentoring
- Coordination avec les equipes metier

## Resultats

- **Performance amelioree** sur les pages critiques
- **Equipe structuree** avec des pratiques modernes
- **Dette technique reduite** progressivement
- **Continuite de service** maintenue pendant les evolutions
